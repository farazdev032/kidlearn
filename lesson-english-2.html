🔊 What is Phonics?

Phonics is learning the sounds that letters make. Every letter has its own special sound. When we know the sounds, we can read and write any word!

💡 Important: The letter NAME and the letter SOUND are different!

The letter B is called "Bee" — but it sounds like the start of Ball!
The letter S is called "Ess" — but it sounds like the start of Sun!
